1.0接口名

Get_data

1.1接口地址

http:///php/api/public/int_mysql_get_data_sheet.php

1.2接口描述

用于获取资料表的全部数据、按UID查询成员数据、按QQ号查询成员数据、获取排序后的资料表、获取非日期格式错误的信息。

1.3请求方式

GET

1.4返回格式

JSON

1.5详细传入参数描述

参数名 默认值 可传入的所有值 / 为分割符 值描述
game_name 无默认值，必要参数 bh3/pcr/ys 崩坏三/公主连结/原神
oder_by serial_asc time_asc/time_des/serial_asc/serial_des 按入群时间升序/按入群时间降序/按序号升序/按序号降序
exclude_error 无默认值，可选参数 exclude 传入该参值后即可排除日期无效的数据
search_qq 无默认值，可选参数 任意的QQ号 QQ号，必须是数值，优先按QQ查找
search_uid 无默认值，可选参数 任意的UID号 UID号，必须是数值，按QQ号与按UID查找资料，二者不可兼得

1.6返回值示例

        [["2019-02-03 00:00:00",
        "1",
        "1658548955",
        "\u4e34\u5189",
        "1067463032368",
        "\u4e34\u5189",
        "\u4e34\u5189",
        "\u4f1a\u957f"]]

1.7返回值详解

参数名/数组下标 值描述
0 该成员入群的时间
1 该条数据在该表的唯一序号
2 该成员的QQ号
3 该成员在群内的昵称
4 该成员在该表下的UID号
5 该成员在游戏内的昵称
6 该成员在群内的惯用称呼
7 该条数据的备注信息
